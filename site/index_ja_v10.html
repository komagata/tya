作業を中断させる小さな選択を取り除くために作られました。

特徴

一意のソース表現

すべてのプログラムには、ただ一つのソース表現があります。

C へのコンパイル

小さくて移植性の高いランタイムと精密なマーク&スイープ GC。仮想マシンも組み込みインタプリタもなし — 配布するのは単一のバイナリだけ。

オールインワンのツールチェイン

tya バイナリにコンパイラ、フォーマッタ、チェッカ、テストランナ、パッケージマネージャがすべて入っています。

親切な診断

すべてのエラーに安定コード (TYA-Exxxx)、ソーススニペット付きバナー、実行可能なヒント、解説へのリンクが付きます。CI とエディタ向けに JSON 出力も。

hello.tya 
user = { name: "komagata", age: 20 }

greet = user -> "Hello, {user["name"]}!"

if user["age"] >= 20
  print(greet(user))

インストール

brew install komagata/tap/tya

Linux 用 tarball をダウンロードし、~/.local 以下にインストールします。 

curl -LO https://github.com/komagata/tya/releases/download/v0.41.0/tya-v0.41.0-linux-amd64.tar.gz
tar xzf tya-v0.41.0-linux-amd64.tar.gz
cd tya-v0.41.0-linux-amd64
./install.sh

Windows 用 tarball をダウンロードし、インストールスクリプトを実行します。 

curl.exe -LO https://github.com/komagata/tya/releases/download/v0.41.0/tya-v0.41.0-windows-amd64.tar.gz
tar.exe xzf tya-v0.41.0-windows-amd64.tar.gz
cd tya-v0.41.0-windows-amd64
powershell -ExecutionPolicy Bypass -File .\install.ps1

実行

  • ビルドして実行 
    tya run examples/hello.tya
  • 実行ファイルをビルド 
    tya build examples/hello.tya -o hello
./hello
