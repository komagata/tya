作業を中断させる小さな選択を取り除くために作られました。

特徴

一意のソース表現

全てのプログラムには、インデントベースのただ一つのソース表現があります。（lossless syntax tree）

配布しやすい実行ファイル

C を経由するのでクロスコンパイルがしやすく、 ランタイム込みの 1 バイナリとして配布できます。 VM や外部インタプリタは不要です。

オールインワンのツールチェイン

tya バイナリにコンパイラ、フォーマッタ、チェッカ、テストランナ、パッケージマネージャがすべて入っています。

hello.tya 
user = { name: "komagata", age: 20 }

greet = user -> "Hello, {user["name"]}!"

if user["age"] >= 20
  print(greet(user))

インストール

Homebrew

brew install komagata/tap/tya

Shell installer

curl -fsSL https://tya-lang.org/install.sh | sh
curl -fsSL https://tya-lang.org/install.sh | sh
irm https://tya-lang.org/install.ps1 | iex

実行

  • ビルドして実行 
    tya run examples/hello.tya
  • 実行ファイルをビルド 
    tya build examples/hello.tya -o hello
./hello

ドキュメント

ガイド 最初のプログラムから日常の構文まで、Tya を学ぶ。 仕様 最新リリースの正確な構文、言語の挙動、組み込み関数、標準ライブラリを調べる。 バージョン 過去リリースの凍結済み文書を見る。 English 英語版ホームに切り替える。
