Easy to write. Easy to read. Friendly to use.

Small language, practical scripts

CoffeeScript feel. Golang practicality.

Tya pairs lightweight, indentation-based writing with straightforward command-line tooling and compile-to-C execution.

hello.tya indented blocks 
user =
  name: "komagata"
  age: 20

greet = user -> "Hello, " + user["name"]

if user["age"] >= 20
  print greet(user)
Install Choose your platform.

Install with Homebrew.

brew install komagata/tap/tya

Download the Linux tarball and install it under ~/.local.

curl -LO https://github.com/komagata/tya/releases/download/v0.16.0/tya-v0.16.0-linux-amd64.tar.gz
tar xzf tya-v0.16.0-linux-amd64.tar.gz
cd tya-v0.16.0-linux-amd64
./install.sh

Download the Windows tarball and run the installer script.

curl.exe -LO https://github.com/komagata/tya/releases/download/v0.16.0/tya-v0.16.0-windows-amd64.tar.gz
tar.exe xzf tya-v0.16.0-windows-amd64.tar.gz
cd tya-v0.16.0-windows-amd64
powershell -ExecutionPolicy Bypass -File .\install.ps1
Quick run Use the repository command-line tool.
  • Build and run tya run examples/hello.tya
  • Build an executable tya build examples/hello.tya -o hello

Language shape

Readable by design, small by constraint.

The page is organized around what the language feels like to read and write.

Blocks

Indented lines form blocks, so nested logic stays visible without extra braces.

Values

Use nil, booleans, numbers, strings, arrays, dictionaries, functions, errors, and modules.

Control

if, elseif, else, while, array loops, dictionary loops, break, and continue.

Functions

Function literals, calls, implicit returns, explicit return, and multiple return values keep scripts compact.

Modules

Split code with module, load it with import module_name, and call members with module.member.
