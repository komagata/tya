from before the Tya v0.1 specification freeze.

It also contains the legacy split self-host compiler and bootstrap scripts:

1. [`selfhost/`](selfhost/)

These files are not current authority for language behavior, standard library
behavior, roadmap planning, or self-host implementation strategy. Use the
current root and docs files instead:

1. [`../../../ROADMAP.md`](../../../ROADMAP.md)
1. [`../../REFERENCE.md`](../../REFERENCE.md)
1. [`../../API.md`](../../API.md)
